# Integrations Overview

To manage integrations, navigate to **Organization Settings** > **Integrations**. The page is organized into three tabs:

- **All:** available integrations you can connect now.
- **Coming Soon:** integrations visible but not yet configurable.
- **Installed:** integrations you have already connected, with health status and configuration details.

Use the search bar to filter by name, description, or tags. Select a provider from the list below for its setup guide.
