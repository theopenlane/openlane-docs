Openlane’s GitHub integration connects your Openlane organization with your GitHub organization, enabling seamless collaboration and automation of security checks within your development workflow.

## Prerequisites
- GitHub organization owner (or repository admin) permissions.
- Openlane organization admin permissions.

## Connect the GitHub App
1. In Openlane, go to Integrations → GitHub → Install.
2. Choose your GitHub organization and select All repositories or Selected repositories.
3. Approve the requested permissions and events.
4. Return to Openlane and confirm the connection shows as Connected.
