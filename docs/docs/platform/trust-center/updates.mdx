# Updates

Share brief announcements about changes to your security practices. Updates appear on your Trust Center and show visitors that you actively manage security.

## What to Announce

- New compliance certifications achieved
- Changes to security policies or practices
- New security features or capabilities
- Responses to industry security events
- Infrastructure updates affecting security

## Character Limit

Updates have a 280-character limit for the description. This encourages concise, focused announcements. For detailed information, link to a blog post or documentation page.

## Examples

**New Certification**
> SOC 2 Type II Certification Achieved - We have completed our annual SOC 2 Type II audit with no exceptions. The report is available in the Documents section.

**Policy Update**
> Updated Data Retention Policy - We have updated our data retention policy with clearer guidelines on data lifecycle management.

**Security Enhancement**
> Multi-Factor Authentication Now Required - All user accounts now require MFA for enhanced account security.

## Best Practices

- Keep updates concise and focused on a single topic
- Post promptly when changes occur
- Use clear, professional language without technical jargon
- Maintain a regular cadence to demonstrate active security management
